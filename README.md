# GitHubRepoDownloader

Download all repositories


## create project

1. git clone
2. go mod init github.com/lemonfisk/GitHubRepoDownloader
3. go run .

## run project

1. Create ssh key for github
2. go run .

## Structure project

- **configs** 
  - **repo.json** (list repositories)
- **downloads**
  - **.gitkeep** 
- **.gitignore**
- **go.mod**
- **main.go**
- **README.md**
